# Сложно сосредоточиться

https://github.com/vilkafork21/slozhno-sosredotochitsya-ad

Адаптивный сайт о концентрации внимания, свёрстанный в рамках проектной работы курса Яндекс Практикум.

## Особенности

- Тёмная и светлая цветовые темы (ручное переключение + системная тема)
- Адаптивная вёрстка: мобильная (375px), планшетная (768px+), десктопная (1024px+)
- Подход Mobile First
- CSS-переменные для цветовых схем
- Grid-раскладка для всех блоков
- Логические CSS-свойства
- Фиксированный фон с `background-attachment: fixed`
- «Ленивая» загрузка изображений в галерее

## Структура

```
├── fonts/          — шрифты IBM Plex Mono
├── images/         — изображения и фавиконки
├── scripts/        — скрипт переключения темы
└── styles/
    ├── globals.css  — сброс браузерных стилей
    ├── variables.css — CSS-переменные (тёмная тема по умолчанию)
    ├── style.css    — основные стили и медиазапросы
    ├── dark.css     — переменные тёмной темы (для принудительного переключения)
    └── light.css    — переменные светлой темы + медиафича
```
